# Voicemail Dashboard

An intelligent voicemail system prototype that transforms unstructured voicemail recordings into structured, actionable work items for clinic admin staff. Built with Next.js.

## Prerequisites

You will need a Google Gemini API Key to run the AI analysis. The application uses the **Gemini 2.5 Flash** model.

Create a `.env` file in the root directory and add your key:

```bash
GEMINI_API_KEY=your_api_key_here
```

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
